Minesweeper

How to play.

Welcome to Minesweeper. If you're a Windows' user chances are that you already have the game on your computer. The rule of the game is simple, the number on a block shows the number of mines adjacent to it and you have to flag all the mines.

You can start by clicking at any random place since it is your first game. Remember, the number is the mines adjacent to the block. If a block with 2 has only two unopened mines flag the unopened ones and same for other numbers (if you got 8, you're a pretty lucky person ;) ).
